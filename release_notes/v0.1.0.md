## QSOT2 v0.1.0 — first release

**QSOT2** (Quantum State Over Time, line 2) is a bounded, deterministic
**mathematical-consistency verifier** for a *phenomenological* quantum-channel
model of curved-spacetime backgrounds. It answers one question and answers it
honestly: *is the model's quantum-mechanical machinery internally consistent and
reproducible, and are its headline numbers honest about what they are?*

This is **model-consistency only** — no external-physical-validity, no
first-principles derivation, no quantum-gravity claim, not peer-reviewed. That
boundary is serialized in every result (`claim_boundary`), not just stated here.

### What this release verifies

Reference run: **`DEGRADED_PASS` — 34 PASS / 0 FAIL / 0 SKIPPED / 1 DEGRADED_PASS**
over 35 checks and six backgrounds (flat, Schwarzschild, de Sitter, AdS₅,
Eguchi-Hanson, Gödel).

| What | Measured |
|---|---|
| Temporal-state axioms (linearity, CPTP completeness, trace preservation, conditionability, density validity) | hold to machine epsilon — all deviations `≤ 4.4×10⁻¹⁶` |
| Per-background purity (curvature → channel `p = 1 − exp(−α·‖Riemann‖_F)`) | Schwarzschild `0.99943`, de Sitter `0.63607`, AdS₅ `0.67764`, Eguchi-Hanson `0.99887` |
| Kirkwood-Dirac flat-relative signal | `kd_delta = +0.1230` (de Sitter − flat) |
| Non-Markovianity proxy (self-tested) | self-test `nm = 1.41×10⁻⁴` (detects injected backflow); model trajectory `nm = 0` (Markovian by construction) |
| Tests | 50 passing, 96% coverage (`≥ 90%` gate), green on CI |

The single degraded check is `kd_optimization_converged` — the flat-baseline KD
optimizer does **not** converge within its fixed 50-step budget. It is reported
as `DEGRADED_PASS`, not rounded up to `PASS`.

### Reproduce it

```bash
git clone https://github.com/Flamehaven-Labs/QSOT2-Compiler
cd QSOT2-Compiler
pip install -e ".[dev]"
python run_experiment.py --config configs/experiment.yaml --out reports
# -> Overall Verdict: DEGRADED_PASS  (Passed: 34 / Failed: 0 / Skipped: 0 / Degraded: 1)
pytest -q
```

Results are deterministic (byte-stable except `generated_at`) and provenance is
anchored to a canonical-LF SHA-256.

### Scope / known limits

- Phenomenological ansatz; `α` is a free calibration parameter, not derived.
- `experimental/` (former `phase6` + Rust sidecar) is excluded from the verdict.
